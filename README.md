# Nyctophobia

Incremental Game 1

Game made by HyproVex

site: https://hyprovex.github.io/Nyctophobia/


# Story Summary 

You woke up in a small room, no bigger than a toilet room. Your memories are blurry. The darkness is making you anxious.
